## Overview

A build helper and simple transpiler for both managing existing C projects and creating new ones. Built with Rust, this aims to fit the philosophy of "Drop your files and it just works".

## Project Scope (Summer MVP)

* Transpilation of simple keywords (e.g. `vari` -> `int`)
* Handling of existing header and CMake files and generating new ones
